# Project Plan

## Purpose

Build a personal fitness tracking web application that helps me monitor my weight loss journey while demonstrating software engineering fundamentals.

## Objectives

- Track daily body weight
- Track daily calorie intake
- Track daily protein intake
- Log workouts
- Display daily progress through a simple dashboard

## Target Users

Primary User:
- Myself

## Technologies

- HTML
- CSS
- JavaScript
- Git
- GitHub

## Project Scope

- Dashboard
- Weight tracking
- Workout logging
- Nutrition tracking
- History page

### Not Included

- AI features
